<h1 align="center">homebrew-catsup</h1>

<p align="center"><strong>Homebrew tap for CatsUp</strong></p>

---

This is the official [Homebrew](https://brew.sh/) tap for [CatsUp](https://github.com/GeiserX/CatsUp).

## Installation

```bash
brew tap geiserx/catsup
brew install --cask catsup
```

Or install directly:

```bash
brew install --cask geiserx/catsup/catsup
```

## What is CatsUp?

CatsUp is a stealthy, cross-platform meeting assistant designed to run locally on your machine. It detects when you join a meeting, records the session, transcribes in real-time, and uses AI to generate summaries and answer questions about the discussion.

**Features:**
- Stealth Mode -- Unobtrusive system tray/menu bar application
- Meeting Detection -- Automatically detects Teams, Zoom, Slack, Google Meet
- Smart Recording -- Captures video, system audio, and microphone
- Real-Time Transcription -- Live speech-to-text with speaker detection
- Screen Vision -- Understands what's on screen using OCR and vision AI
- AI-Powered Responses -- Contextual answers using cloud or local models
- Private Options -- Use cloud APIs or 100% local processing

## Requirements

- macOS 13 (Ventura) or later

## Uninstall

```bash
brew uninstall --cask catsup
```
